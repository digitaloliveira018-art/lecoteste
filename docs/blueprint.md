# **App Name**: Leco Fretes

## Core Features:

- Interativo Budget Quiz: A multi-step, mobile-first questionnaire to gather trip details like volume, stairs, and locations.
- AI Space Advisor Tool: An intelligent tool that analyzes the text description of the items to transport and recommends the ideal vehicle size for the job.
- WhatsApp Automation: Automatically generates and formats a detailed budget request message and directs the user to the company WhatsApp with all details encoded.
- Dynamic Service Display: Responsive visual cards showcasing service types like small moves, freight, and intercity trips.
- Progress Tracking: Visual progress bar during the scheduling process to keep users engaged and informed on /agendar.
- Customer Feedback Section: Display area for verified testimonials to build trust with new local customers.

## Style Guidelines:

- Primary color: Vibrant Yellow (#FFCC00) chosen for energy and visibility, used for headers and primary backgrounds.
- Background color: Soft Cream (#FFFDF0) a desaturated, high-brightness variation of the primary yellow to ensure readability.
- Accent color: Intense Logistics Blue (#0046B3) for buttons and icons to convey reliability and professionalism.
- Headline font: 'Poppins' (Sans-serif) for a precise, geometric, and strong contemporary feel. Body font: 'Inter' (Sans-serif) for high legibility and an objective look across devices.
- Bold, flat icons in blue featuring delivery trucks, cardboard boxes, map pins, and WhatsApp symbols.
- Clean, structured block layout with high contrast between yellow and blue sections; optimized for mobile interaction.
- Smooth, subtle horizontal transitions between steps in the quiz flow to reduce cognitive load.
